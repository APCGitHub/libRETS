Introduction

libRETS is RETS client library written in C++ that allows rapid development of RETS client applications. By saving the developer from dealing with the RETS protocol details, the application writer can concentrate on their application, saving them time and money in the process. It is released under a BSD-style license.

Features

  • Provides access to login, logout, search, and get object.
  • Can connect to RETS 1.0, 1.5, 1.7, 1.7.2, and 1.8 compliant servers.
  • Can be used on many platforms, including Visual C++ on Windows, and gcc on just about any flavor of Unix.
  • Native bindings for .Net, Java, Perl, PHP5, Python, and Ruby.

The current stable version is 1.6.1. View the complete version history.
